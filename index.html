Quiz Instructions

Welcome to the Block Based coding Quiz! Read the instructions carefully before you begin:

  • Answer each question to the best of your ability.
  • Select only one option for each question.
  • Some questions might include code snippets or block images.
  • Click "Next" to go to the next question.
  • Click "Submit" on the last question to see your score!
Note
  • You can not go to next question without solving the current question
  • You can not go to the question. There is no back button.
  • So Click the 'Next Button' only when you are sure of the current question
  • You must choose at least one of the options, even if you don't know the answer. Otherwise you will not be able to go to the next question.
